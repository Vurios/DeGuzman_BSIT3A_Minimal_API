A C# ASP.NET Core Minimal API project demonstrating streamlined backend development, RESTful routing, and database migration management. This repository serves as a practical implementation of modern web API architecture.

## 📖 Overview
This project was developed as part of the Bachelor of Science in Information Technology (BSIT) curriculum for Block 3A at Bicol University Polangui. It highlights the use of C# and .NET to create lightweight, high-performance web services. The architecture focuses on minimizing boilerplate code while ensuring scalable database interactions and seamless initial commit migrations.

## 💻 Tech Stack
* **Language:** C#
* **Framework:** ASP.NET Core (Minimal API)
* **Data Access:** Entity Framework Core (with automated migrations)
* **Solution Format:** `.slnx`

## ✨ Features
* **Minimal API Routing:** Streamlined HTTP GET, POST, PUT, and DELETE request handling.
* **Database Migrations:** Configured for seamless Entity Framework code-first database updates and schema management.
* **Clean Solution Architecture:** Organized directory structure for easy maintainability and future expansion.

## 🚀 Getting Started

### Prerequisites
* [.NET SDK](https://dotnet.microsoft.com/download) (Version 7.0 or 8.0 recommended)
* Visual Studio 2022, JetBrains Rider, or VS Code
* SQL Server or a compatible local database provider
